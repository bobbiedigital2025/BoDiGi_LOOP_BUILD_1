## 🚀 Getting Started (3 Steps)

### Step 1: Open the Editor
```bash
# Option A: Double-click pdf-editor.html (opens in default browser)

# Option B: Start local server (recommended)
python3 -m http.server 8000
# Then open: http://localhost:8000/pdf-editor.html
```

### Step 2: Load a PDF
- Click "📁 Open PDF" button
- Select a PDF file (or use test-sample.pdf)
- PDF will display in the editor

### Step 3: Start Editing
- Click "✏️ Add Text" to add text
- Click "⬜ Add Rectangle" to draw shapes
- Click "💾 Save PDF" to download

**That's it!** You're editing PDFs!
