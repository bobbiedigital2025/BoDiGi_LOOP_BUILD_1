## 📄 PDF Editor

A professional, browser-based PDF editor is now available! Edit, annotate, and enhance your PDF documents without any server-side processing.

**[Open PDF Editor](pdf-editor.html)** | **[View Documentation](PDF_EDITOR_README.md)**

### Features
- ✏️ Add custom text to PDFs
- 🎨 Draw shapes and rectangles
- 💾 Save edited PDFs with all modifications
- 🔒 100% client-side processing (secure & private)
- 📱 Responsive design for all devices

### Quick Start
1. Open `pdf-editor.html` in your web browser
2. Click "Open PDF" to load a document
3. Use the toolbar to add text or shapes
4. Click "Save PDF" to download your edited document

For detailed instructions and features, see [PDF_EDITOR_README.md](PDF_EDITOR_README.md).
