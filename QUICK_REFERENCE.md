# Quick Reference Card - BoDiGi PDF Editor

## 🚀 Quick Start
```
1. Open pdf-editor.html in browser
2. Click "Open PDF" button
3. Select your PDF file
4. Start editing!
```

## 🎨 Toolbar Buttons

| Button | Function | Shortcut |
|--------|----------|----------|
| 📁 Open PDF | Load a PDF file | - |
| 💾 Save PDF | Download edited PDF | - |
| ✏️ Add Text | Enable text mode | - |
| ⬜ Add Rectangle | Enable shape mode | - |
| 🗑️ Clear All | Remove page annotations | - |

## ✏️ Text Mode

**Activate**: Click "Add Text" button

**Properties**:
- Text Content: What to write
- Font Size: 8-72px
- Text Color: Any color

**Usage**: Click on PDF to place text

## 🎨 Rectangle Mode

**Activate**: Click "Add Rectangle" button

**Properties**:
- Border Color: Outline color
- Fill Color: Inside color
- Border Width: 1-10px

**Usage**: Click and drag to draw

## 📄 Page Navigation

| Control | Function |
|---------|----------|
| ← Previous | Go to previous page |
| Next → | Go to next page |
| Page X of Y | Current page indicator |

## 💾 Saving

1. Click "Save PDF" button
2. File downloads as `edited-document.pdf`
3. All annotations are saved permanently

## ⚙️ Settings Panel

Located in the left sidebar:

### Text Properties
- **Text Content**: Type your text
- **Font Size**: Number input (8-72)
- **Text Color**: Color picker

### Shape Properties
- **Border Color**: Outline color picker
- **Fill Color**: Fill color picker
- **Border Width**: Number input (1-10)

## 🎯 Common Tasks

### Sign a Document
```
1. Open PDF
2. Click "Add Text"
3. Type your name
4. Choose large font size
5. Click to place signature
6. Save PDF
```

### Highlight a Section
```
1. Open PDF
2. Click "Add Rectangle"
3. Choose bright border color
4. Set fill color to transparent/white
5. Draw around the section
6. Save PDF
```

### Fill a Form
```
1. Open form PDF
2. Click "Add Text"
3. Set appropriate font size
4. Click in each field
5. Type the information
6. Save PDF
```

### Add Comments
```
1. Open PDF
2. Click "Add Text"
3. Type your comment
4. Choose a color (red for important)
5. Place near relevant content
6. Save PDF
```

## 🔧 Troubleshooting

### PDF Won't Load
- ✅ Check file is valid PDF
- ✅ Verify file isn't password-protected
- ✅ Try different browser

### Can't See Annotations
- ✅ Check mode is active (button highlighted)
- ✅ Verify correct page is selected
- ✅ Ensure properties are set

### Save Not Working
- ✅ Allow pop-ups in browser
- ✅ Check download permissions
- ✅ Try different browser

## 💡 Tips

### Text Tips
- Use 16-20px for normal text
- Use 24-32px for headers
- Dark colors are more readable
- Keep text concise

### Shape Tips
- Bright colors draw attention
- White fill preserves content
- Thicker borders are more visible
- Draw precisely for best results

### Workflow Tips
- Save frequently
- Test with sample PDF first
- Use consistent colors
- Preview before finalizing
